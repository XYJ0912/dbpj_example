# 猫咪日常健康与行为记录表

适用于长期饲养中的饮食、排泄、体重和行为变化追踪

## 1. 记录目标

这份记录表用于发现猫咪状态变化，不用于替代兽医诊断。重点是把“偶尔异常”和“持续异常”区分开。

## 2. 每日记录

| 项目 | 正常记录方式 | 需要关注的变化 |
|---|---|---|
| 食欲 | 记录主食摄入量和剩余量 | 明显少吃、拒食、突然暴食 |
| 饮水 | 观察水碗或饮水机消耗 | 饮水突然增多或明显减少 |
| 排尿 | 记录尿团数量和大小 | 频繁进出猫砂盆、尿团变小、无尿 |
| 粪便 | 记录形态、颜色、频率 | 软便、腹泻、便秘、带血 |
| 精神 | 记录活跃度和互动意愿 | 躲藏、嗜睡、攻击性突然增加 |

## 3. 每周记录

- **体重**：固定时间称重，例如每周日上午。
- **毛发和皮肤**：观察掉毛、结痂、皮屑、局部脱毛。
- **口腔和眼鼻**：注意口臭、流口水、眼部分泌物、打喷嚏。
- **运动与跳跃**：记录是否回避跳高、跛行或动作迟缓。

## 4. 行为变化拆解

猫咪行为变化通常与环境、健康、资源分布和互动方式有关。

| 行为 | 可能变量 | 先检查什么 |
|---|---|---|
| 乱尿 | 压力、泌尿问题、猫砂盆不合适 | 排尿频率、猫砂盆位置、近期环境变化 |
| 过度舔毛 | 皮肤不适、压力、疼痛 | 皮肤状态、跳蚤、舔毛部位 |
| 夜间吵闹 | 精力未释放、饥饿、作息错位 | 睡前互动、喂食时间、白天活动量 |
| 攻击人 | 恐惧、疼痛、玩耍边界不清 | 触碰部位、攻击前信号、互动方式 |

## 5. 就医前整理

- 异常开始的日期和持续时间。
- 最近 3-7 天的食欲、饮水、排泄记录。
- 近期是否换粮、搬家、添置新宠或改变猫砂。
- 拍摄呕吐物、粪便、异常步态或异常行为的视频。
